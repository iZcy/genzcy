# GenZCY — Private Class Flow

## 8 Sessions Overview

| Ses | Title | Core content |
|---|---|---|
| **1** | **Introduction to Vibe Coding Responsibly** | Compare Campus vs Practicals vs Vibe Code. Dev history → OpenCode + PromptAudit. Zero-level vanilla stack (HTML/CSS/JS, JS backend, SQLite). Deploy to Netlify via GitHub. |
| **2** | **The Prompt Architect** | AI history (expert systems → LLM). Context management (Markdowns, MCPs, Skills). SDLC + Blackbox Dev (error-driven + test-driven). Clean code, clean repo. Fork repo → 2 projects → deploy via push MCP → Netlify. |
| **3** | **Do Not Reinvent The Wheels** | Why frameworks exist. Our stack: Vite+TailwindCSS, Express, MongoDB. Complete beginner dev track. Deploy framework app. |
| **4** | **Trust, But Verify** | pino structured logging. Break case (intro bug → trace → fix). Analytics (Plausible/GA). Business case (simulate visitor → capture analytics). Log viewer + analytics dashboard. |
| **5** | **From Click to Cash** | Booking + payment end-to-end. Webhook + email receipt. Break: webhook failure. |
| **6** | **Know Thy Customer** | JWT auth + dashboard + roles. Email notifications. Break: role leak. |
| **7** | **Ready, Set, Ship** | SEO + performance + security + compliance. Break: mixed content. |
| **8** | **Full Circle** | Solo free build. PromptAudit review. Community showcase. Handoff conversation. |

## Session Rhythm (120 min)

```
00-05  Check-in, review PromptAudit feedback
05-15  Concept — business analogy
15-55  BUILD — owner prompts, AI builds
55-85  READ — owner reads every changed file
85-100 Error — break case, trace + fix together
100-110 Commit + push (owner runs git)
110-120 Preview next, update Build Journal
```
