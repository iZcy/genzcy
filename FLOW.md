# GenZCY — Private Class Flow

## Session Overview

| Ses | Title | What they build | Key business need |
|---|---|---|---|
| **1** | **"My First Page"** | HTML page + git init + first commit/push | "I have a repo" |
| **2** | **"Brand + Contact"** | Branded layout + contact form → email | "Customers can reach me" |
| **3** | **"Content + SEO"** | Services page + FAQ + SEO tags + sitemap | "Google can find me" |
| **4** | **"Live Data"** | Dynamic pages reading from DB + admin edit | "My content is alive" |
| **5** | **"Bookings + Maps + Payments"** | Booking flow + payment gateway + Maps embed | "Customers can pay me" |
| **6** | **"Logins + Dashboard"** | Login + admin dashboard + order mgmt | "I control my site" |
| **7** | **"Deploy + Measure"** | Live deploy + HTTPS + analytics + 404 page | "Site is live & fast" |
| **8** | **"Solo + Community"** | Owner builds alone from scratch | "I can do this myself" |

## Session Rhythm (120 min)

```
00-05  Check-in, review last PromptAudit feedback
05-15  Concept — business analogy for today's layer
15-55  BUILD — owner prompts, AI builds, code appears
55-85  READ — owner reads every changed file, asks why
85-100 Error — something breaks, trace + fix together
100-110 Commit + push (owner runs git commands)
110-120 Preview next session, update Build Journal
```
